## 🔧 API Компонентов

### ButtonMap
```tsx
<ButtonMap 
  onClick={() => {}}
  text="Нажми меня"
  position="top-right"          // 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'
  disabled={false}              // отключить кнопку
  isLoading={false}             // показать спиннер с текстом "⏳ Загрузка..."
  style={{ opacity: 0.8 }}      // доп. стили
/>
```

### LoadingSpinner
```tsx
<LoadingSpinner 
  size="medium"                 // 'small' | 'medium' | 'large'
  label="Загружаем кафе..."     // опциональный текст
/>
```

### useCoffeeShopsInView
```tsx
const { 
  coffeeShops,                  // массив кофейней в текущем виде карты
  isLoading,                    // идет ли загрузка
  error,                        // ошибка если есть
  boundsChanged,                // границы значительно изменились?
  setBoundsChanged              // функция сброса флага
} = useCoffeeShopsInView()
```

## 🎯 Поведение приложения

1. **При загрузке карты**: 
   - Область карты определяется, кафе загружаются
   - Показывается спиннер в центре

2. **При движении карты**:
   - После остановки (moveend + debounce 1500ms) проверяется величина изменения
   - Если изменение значительно → запросятся новые кафе  
   - Включится спиннер
   - Появится кнопка "Найти ☕ в этом районе"

3. **Кнопки на карте**:
   - ТОП-ПРАВО: "🔍 Ближайшее кафе" - всегда видна, загружается список
   - НИЗ-ПРАВО: "☕ Найти в этом районе" - появляется при изменении границ

## 💡 Оптимизация

- ✅ Debounce запросов (1500ms) - избегаем частых запросов при скроллинге
- ✅ React Query кэширование (10 минут staleTime, 15 минут gcTime)
- ✅ Проверка минимального изменения границ (0.005 градусов)
- ✅ Мемоизация компонента CoffeeShopsLoader
- ✅ useCallback для обработчиков событий
